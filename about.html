About

Ghost in the Bit is a personal project — part field notebook, part teaching tool. It collects observations from offensive security work and turns the recurring patterns into practice scenarios that anyone can run in their browser.

Why it exists

Most investigators learn the craft on production data they can't talk about, in tools they can't show off. That leaves a gap: there's no shared, public, realistic environment to practice traffic analysis in. Ghost in the Bit tries to be that environment — with synthetic data, real signal coverage, and the analytical reasoning written out so the learning is reproducible.

What you'll find here

  • The Phantom Feed platform — a browser-based SQL workbench with scenario-driven investigation training. Pick a scenario, get a synthetic HTTP log dataset, and work the case.
  • The blog — short, practical writeups of real techniques (rate-limit bypasses, fingerprinting, host-header attacks, sandbox escapes), with identifying details changed for client confidentiality.
  • The generator — the open-source Python tool used to produce the platform's scenarios, so you can see how realistic adversarial datasets are constructed. Build your own.
